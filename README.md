# Easyx-

基于图形库Easyx实现的网络双人五子棋  可以局域网对战，若将服务器代码部署到云服务器，可以实现跨局域网对战

# 简单说两句

IP地址部分请自行转换成自己服务器的IP地址

有部分小bug，且鼠标按键不灵敏
